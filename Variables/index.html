Variables in JavaScript

A JavaScript Variable is just a name of storage

And we can give that name in three ways

  • Using let Keyword
  • Using const Keyword
  • Using var Keyword

Rules for naming Variables.

A variable name can conatain any of the following character but must not start with a digit.

  • Capital Letters: A to Z
  • Small Letters: a to z
  • Digits: 0 to 9 (but we can't start variable name with digit)
  • Special Character: we only use two specail characters that is underscore(_) and doller(&) between the letters

We define variable by three cases like:

  • Camelcase: In camelcase we write a varible like this 'userIntro'
  • Snakecase: In this we write a variable in lower case like 'userintro'
  • Underscore: In this case we user underscore between the letters
  • Kebab case: we use hypen(-)instead of underscore
  • Pascal case: start with capital letter. in case of multiple word so all word start with capital letter
