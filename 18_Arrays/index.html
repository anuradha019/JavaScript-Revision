Basic Array methods

  • pop()

    • This method is used for remove element from the last of arrey

    Example: const fruitsCollection = ['mango', 'apple', 'grapes']

    fruitsCollection.pop()

    output:['mango', 'apple']

  • push()

    • This method is used for add element at the end of the arrey

    Example: const fruitsCollection = ['mango', 'apple', 'grapes']

    fruitsCollection.push('papaya')

    output:['mango', 'apple', 'grapes', 'papaya']

  • shift()

    • This method is used for remove element from the start

    Example: const fruitsCollection = ['mango', 'apple', 'grapes']

    fruitsCollection.shift()

    output:['apple', 'grapes']

  • unshift()

    • This method is used to add element at the starting of the arrey

    Example: const fruitsCollection = ['mango', 'apple', 'grapes']

    fruitsCollection.unshift('papaya')

    output:['papaya', 'mango', 'apple', 'grapes', 'papaya']

  • concat()

    • This method is used to add more arrey and create new arrey where both arrey in a single array

    Example: const fruitsCollection = ['Mango', 'Apple', 'Grapse', 'Papaya']

    const vegitables = ['Potaoes', 'Okra', 'spinich']

    fruitsCollection.concat(vegitables)

    Outputs: ['Mango', 'Apple', 'Grapse', 'Papaya', 'Potaoes', 'Okra', 'spinich']

  • indexof()

    • This method is used to show perticular data with the help of theire index number

    Example: const fruitsCollection = ['Mango', 'Apple', 'Grapse', 'Papaya']

    fruitsCollection.indexof(3)

    Output: Papaya

  • includes()

    • In this method we know the element in there or not it give output in boolean form

  • reverse()

    • This method is used to reverse all the element of the array

    Example: const fruitsCollection = ['Mango', 'Apple', 'Grapse', 'Papaya']

    fruitsCollection.reverse()

    Output: ['Papaya','Grapse', 'Apple', 'Mango']

  • sort()

    • This method is used toh arrange element in alphabatical order

    Example: const fruitsCollection = ['Mango', 'Apple', 'Grapse', 'Papaya']

    fruitsCollection.sort()

    Output: [ 'Apple', 'Grapse', 'Mango', 'Papaya']

  • slice()

    • This method is used to show element from the given their index number and it gives new arrey and we also give the end number

    Example: const fruitsCollection = ['Mango', 'Apple', 'Grapse', 'Papaya']

    fruitsCollection.slice(1)

    Output: ['Apple', 'Grapse', 'Papaya']

    Example 2: const fruitsCollection = ['Mango', 'Apple', 'Grapse', 'Papaya']

    fruitsCollection.slice(1, 3)

    Output: ['Apple', 'Grapse']

  • splice()

    • This method is used to delete element from arrey according to given index number and it modified the arrey not creat new arry and jo humne delete kiya hai uske badle new item bhi sath me add kar sakte hai

    Example: const fruitsCollection = ['Mango', 'Apple', 'Grapse', 'Papaya']

    fruitsCollection.slice(2, 1, 'Lichi') 2 for index number and 1 ki kitne item delete krna hai

    Output: ['Mango', 'Apple', 'Lichi', 'Papaya']
